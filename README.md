# E-commerce Sales & Pricing Analysis

This project demonstrates a simple end-to-end data analysis workflow using Python and Power BI.

# Tools Used
- Python (data cleaning and basic analysis)
- Power BI (KPIs and dashboards)

# What was done
- Cleaned raw e-commerce transaction data in Python
- Created basic metrics such as total sales, total orders, and average discount
- Built a Power BI dashboard to analyze category-wise and monthly sales trends

# Outcome
This project shows how cleaned data can be converted into meaningful business insights using dashboards.
